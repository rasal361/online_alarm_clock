# ⏰ CHRONO — Online Time Suite

A sleek, dark-themed browser-based time utility featuring a Clock, Timer, Alarm, and Stopwatch — all in one lightweight HTML file with zero dependencies.

## ✨ Features

### 🕐 Clock
- Live real-time digital clock
- Full date display (day, month, year)
- Animated seconds progress bar

### ⏱ Timer
- Set custom hours, minutes, and seconds
- Animated circular countdown ring
- Urgent pulse animation in the last 10 seconds
- Audio alert when the timer ends
- Pause, resume,and reset support

### 🔔 Alarm
- Set multiple alarms with custom labels
- Toggle individual alarms on/off
- Visual flash + audio alert when an alarm fires
- One-click dismiss

### ⏩ Stopwatch
- Millisecond precision timing
- Lap tracking with best/worst lap highlighting
- Pause, resume, and reset support

## 🚀 Getting Started

No installation or setup needed. Just open the file in any modern browser.

### Option 1 — Open Locally
```bash
git clone https://github.com/rasal361/online_alarm_clock.git
cd online_alarm_clock
# Open index.html in your browser
```

### Option 2 — Live Demo
Deploy via [GitHub Pages](https://pages.github.com/) for instant access from any device.

> Go to **Settings → Pages → Source → main branch → / (root)** and your app will be live at:
> `https://rasal361.github.io/online_alarm_clock/`
